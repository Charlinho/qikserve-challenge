# QikServe Challenge

## Requirements:
* Docker
* Java 8

### Running Wiremock

In root directory run command below:

```bash
  docker-compose up
```

### Running App

```bash
  mvn spring-boot:run
```

After run commands above, check if server is running then follow next step

### Swagger

Use the below link to access swagger:

http://localhost:8081/swagger-ui.html
